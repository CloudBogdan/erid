# Erid

## Docs.
`m`, `p` and `gap` - `margin` and `padding`
1 - 5 - levels of `margin`, `padding` or `gap` (example: `m2` = `margin: 20px`)
`r`, `l`, `t`, `b`, `v`, `h` - direction of `margin` or `padding`
